# Contributing

The contributing guide lives in the [`contributing/`](contributing/)
folder, split by task:

- [`contributing/README.md`](contributing/README.md) — start here: how a
  request flows, where contracts live, shared conventions, definition of
  done.
- [`contributing/new-feature.md`](contributing/new-feature.md) — a
  complete new feature, storage to frontend (the real `/currencies`
  endpoints, step by step).
- [`contributing/feature-without-infra.md`](contributing/feature-without-infra.md)
  — a feature composing existing storage; what to skip and why.
- [`contributing/update-route.md`](contributing/update-route.md) —
  changing an existing endpoint (the real `from`/`to` filter diff).
- [`contributing/bug-fix.md`](contributing/bug-fix.md) — reproducing,
  testing, and fixing at the right layer.
